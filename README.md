1. Replace `YOUR_WORKER_NAME.YOUR_ACCOUNT.workers.dev` in `script.js`.
2. Replace `YOUR_SECRET_TOKEN` with the same token used in the Worker.
3. Enable GitHub Pages for this repo.
4. Deploy the Worker using the file below.
